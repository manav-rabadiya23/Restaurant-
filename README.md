# 🍝 Bella Cucina - Italian Restaurant Website

A modern, responsive restaurant website showcasing authentic Italian cuisine with elegant design and smooth user experience.

![Restaurant Website](https://images.unsplash.com/photo-1768697358705-c1b60333da35?crop=entropy&cs=tinysrgb&fit=max&fm=jpg&q=80&w=1200)

## 🚀 Quick Start

The website is built with React, TypeScript, and Tailwind CSS and is ready to preview in your browser.

## ✨ Features

- 🎨 **Modern Design**: Dark theme with elegant gold accents
- 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop
- 🍽️ **Food-Focused**: High-quality food photography
- ⚡ **Smooth Interactions**: Hover effects and animations
- 📋 **Complete Sections**: Hero, Menu, About, Contact

## 📂 Project Structure

```
src/app/
  ├── App.tsx                 # Main app component
  └── components/
      ├── Navigation.tsx      # Header & navigation
      ├── Hero.tsx            # Landing section
      ├── Menu.tsx            # Menu with dishes
      ├── About.tsx           # Restaurant story
      └── Contact.tsx         # Contact form
```
